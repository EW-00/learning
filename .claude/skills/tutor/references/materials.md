# 循环 A · 开学流程：从「我想学 X」到第一课

目标：一次会话内走完全部六步，当天上第一课。不要把开学变成漫长的问卷。

## 第 1 步 · Purpose 访谈（≤5 个问题，够用就停）

必问：
1. **这次学它是为了什么？** 学完你想能独立做到的那件事是什么？（答案将成为终点任务）
2. **要到什么深度？** 三档：`速成应用`（几小时–几天，够用就好）/ `系统深学`（数周–数月，不留漏洞）/ `兴趣考级`（按考纲走）
3. **时间预算与 deadline？**（每周大约几次、每次多久、有没有硬截止）

选问（有疑问才问）：
4. 已有哪些相关背景？
5. 这个科目想用中文还是英文学？（术语一律保留教材原文）

答案写入 `tracks/<slug>/CHARTER.md`（模板：`templates/CHARTER.md`）。slug 用英文小写连字符，如 `music-theory`、`mip`。

## 第 2 步 · 教材审读与策展

三种进料模式，按她给的信息选：

- **模式 ①（默认）· 家教策展**：派 2–3 个并行 research agent 调研该领域候选教材/课程（经典教材、近 3 年新教材、优质公开课各至少一路），每本按四个标准审读：**时效性**（内容是否过时）、**易读性**（写作质量、是否适合自学）、**purpose 匹配度**（速成要的是「围绕真实任务的最小课程」，不是权威大部头）、**练习质量**（有没有可用的习题）。结论写入 `SOURCES.md`：选哪本为主、哪几章跳过、用哪本的哪部分补、为什么。
- **模式 ② · 跟课**：她丢来一本书/一门课（如 MIT OCW 链接），家教读完材料的目录与结构，roadmap 跟随其节奏，只做裁剪（按 purpose 砍 nice-to-know）和补强（薄弱处补其他材料）。
- **模式 ③ · 多本为基**：她指定几本书，家教审读后做「取长」方案：哪个主题用哪本讲，写明理由。

**硬规则**：课程结构必须有教材或公认大纲背书（护栏 10）。找不到合适材料的小众领域，明示「以下大纲是我基于 X、Y 综合的，无单一教材背书」，并把风险说清。

### 教材落地（定稿后立即做）

1. **取得本体**：能合法获取的（官方免费 PDF、公有领域、开放课程讲义、她已购买的电子书由她放入），存到 `tracks/<slug>/materials/`。获取不到本体的（纸书、付费课），在 SOURCES.md 清单里标注「无本地副本」，上课以教学摘要为准。
2. **登记清单**：`SOURCES.md` 的「本地文件清单」表——文件名、内容、来源 URL、许可。`materials/` 整个目录被 gitignore（repo 是 public 的，版权材料不入库；开放许可的材料可例外地移到 `notes/` 旁提交）。
3. **生成教学摘要**：审读每份材料时，把与 ROADMAP 各模块对应的核心内容蒸馏成 `tracks/<slug>/notes/<module>-digest.md`（家教原创产物，**入库**）。摘要标准：包含该模块的关键定义、核心例子、易错点、练习题源指针；一个模块一份，开学时先做前 1–2 个模块的，其余随进度补。
4. **跨机器语义**：git 同步的是清单 + 摘要，不是教材本体。在 firm laptop 上开课时：有本体最好，没有就按摘要教学；需要本体时按清单的来源 URL 重新下载。

## 第 3 步 · Backward Design 路线图

顺序不可颠倒（Wiggins & McTighe）：

1. **定终点任务**：学完能独立完成的真实任务（来自访谈问题 1），写成可观察的行为（「能独立为一个 20 变量的排产问题建 MIP 模型并解释 solver 日志」，不是「掌握 MIP」）。
2. **先设计终点考核**：终点任务的考核形式 + rubric 草案（详见 `assessment.md`）。
3. **倒推先修依赖链**：从终点目标反向展开需要的知识组件，构成依赖图。
4. **裁剪**：`速成应用` 模式砍掉一切不在依赖链上的内容——速成是围绕真实任务重新裁剪，不是压缩版全课程；`系统深学` 保留完整链条；`兴趣考级` 按考纲对齐。
5. **组织模块梯**：由简到繁的完整任务序列（4C/ID），每模块写：可观察目标（禁用「了解/理解/掌握」）、对应教材章节、过门标准、预计时长。

写入 `ROADMAP.md`（模板：`templates/ROADMAP.md`）。

## 第 4 步 · 自适应诊断（有背景才做，零基础跳过）

沿依赖图从中等难度出题，知识空间剪枝：
- 答对一题 → 其全部先修标记为「大概率已会」，向更难处探
- 答错一题 → 其全部后继标记为「未会」，向更易处探
- 目标：≤20 分钟、≤15 题定位知识边界；已会模块在 ROADMAP 标 `✅ 免修`，但其核心概念仍入卡片库（低频复习验证）

诊断结论（含证据）写入 `STATE.md`。

## 第 5 步 · 节奏承诺

成人自学最大死因是拖延弃学（PSI 教训）。和她约定：
- 每周节奏（如「工作日 2×15 分钟复习 + 周末 1×60 分钟深学」），写入 CHARTER
- 节奏是承诺不是枷锁：出差周提前降档（见 `motivation.md` 的 if-then 计划），完不成不清算

## 第 6 步 · 第一课

当场按 `pedagogy.md` 开讲。第一课刻意小而完整：一个概念 + 一次成功的独立练习 + 3–5 张卡片，让她带着「已经开始了」的事实离开。
