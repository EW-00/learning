# 今日复习包 · 2026-07-23

> 只有问题，没有答案。凭记忆作答，答完回复家教或开一次学习会话来判分。预计 5 分钟。

**1.** 商朝管理地方靠「外服」，周朝靠「分封」，这两套看似都是"中央管地方"的办法，本质差别落在哪一步上？周为什么非要从前者升级到后者？

**2.** 一个刚打下天下的小邦，疆域一下子大到自己鞭长莫及。西周当年选了"分封"这条路——它当下解了什么燃眉之急？又给三百年后埋了什么雷？

**3.** 有人质疑：立继承人为什么不挑最能干的儿子，非要死板地传给正妻长子？请说出这个"死板"标准背后真正想买到的东西（用"决策成本 / 争位"的角度），以及它如何被层层复制成一整张网。

**4.** 都说西周初年那场继承危机不是凭空冒出来的，根子在前一个王朝的规矩上。商朝的王位究竟怎么传？为什么这套传法会给"传弟还是传子"留下模糊地带？

**5.** 用现代"机制设计 / 激励对齐"的语言看，儒家和法家对人性走的是两条相反的路：一条想 ___ 人性，一条想 ___ 人性。各填一个动词，并说清法家那条路具体怎么让个体的自利行为反而产出对国家有利的结果。

**6.** 儒家讲的爱是"先亲后疏"的。墨家偏说这种分亲疏的爱恰恰是天下大乱的病根。墨家开的药方叫什么（那个核心词）？由它能顺带推出他们哪几条具体主张（至少说两条）？

**7.** 别看笔记——按时间顺序把学到现在的朝代 / 时期连着报一遍，各配大致起止年代：夏 → 商 → 西周 → 东周（春秋 / 战国）→ 下一个是谁、哪一年？
